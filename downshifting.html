Downshifting Pastime

2019 / Studio Project / Individual

Downshifting Pastime is a board game designed to promote a healthy downshifting lifestyle to Hong Kong citizens through a fun and interactive method.

The game kit contains:
- 40 activity cards
- a set of 4 character pieces
- 40 tokens
- a game board
